# 🐞 Issue Tracker Lite

A simple full-stack issue tracker built with:

- **Backend:** .NET 8 + SQLite + EF Core
- **Frontend:** Next.js + Tailwind + TypeScript
- **Design:** Swagger-inspired dark UI

## 🧱 Running locally

### Backend
```bash
cd IssueTracker.Api
dotnet run
```

### Frontend
```bash
Copy code
cd frontend
npm install
npm run dev
```
App runs on http://localhost:3000
